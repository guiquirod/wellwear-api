# Wellwear

El API de 'WellWear' fue realizada por Guillermo Quintanilla Rodrigo para el Master Universitario de Aplicaciones y Sitios Webs, de la UOC.

El estado en el que se encuentra ahora el proyecto en este repositorio corresponde al de la versión que se encuentra ahora en produccion.

Para realizar las pruebas en local previas a la subida a produccion, se utilizó XAMPP (Apache y MyQSL). Si se quisiera volver a probar, sería necesario modificar manualmente los datos de conexión a la base de datos en el archivo 'db_config'.

Creando una base de datos de nombre 'wellwear' con las tablas del archivo SQL presentes en el repositorio, y moviendo los archivos del API en xampp/htdocs, ya se podría ejecutar el API.
